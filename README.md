# Logic Box

- A beginner-friendly Python project that generates star patterns and analyzes a range of numbers by identifying even and odd numbers while calculating their total sum. This project helps beginners understand loops, conditional statements, nested loops, arithmetic operations, and user interaction in Python.

## Project Objective

- The main goal of this project is to help beginners understand how Python handles:

- User input
- Conditional statements (if, elif, else)
- Loops (while and for)
- Nested loops
- Range functions
- Arithmetic operations
- - Pattern generation
- Even and odd number checking

## Features

- Displays a menu-driven program
- Generates a star (*) pattern
- Checks whether numbers are even or odd
- Calculates the sum of numbers in a given range
- Uses loops for repeated execution
- Beginner-friendly and easy to understand

## Program Workflow

1. Display the welcome message
2. Show the menu options:

   * Generate a pattern
   * Analyze a range of numbers
   * Exit the program
3. Take user input for menu selection
4. Perform the selected operation:

   * Generate a star pattern using nested loops
   * Check even/odd numbers and calculate total sum
5. Continue running until the user selects Exit

# Concepts Used

- while loop
- for loop
- Nested loops
- Conditional statements
- Modulus operator (%)
- User input using input()
- Integer conversion using int()
- Formatted strings (f-strings)
